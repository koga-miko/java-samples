# コールバック・ラムダ式・ストリーム

## コールバック
### 伝統的な方法：インターフェース
```java
// Javaでコールバックを実現する伝統的な方法 = インターフェース
interface OnComplete {
    void done(String result);
}

void doSomething(OnComplete callback) {
    // 処理...
    callback.done("完了しました");
}

```

### 匿名クラスで渡す
```java
doSomething(new OnComplete() {
    @Override
    public void done(String result) {
        System.out.println(result);
    }
});
```

### ラムダで書き直す
```java
// メソッドが1つだけのインターフェース（関数型インターフェース）なら
// ラムダで書ける
doSomething(result -> System.out.println(result));

// さらにメソッド参照でも書ける
doSomething(System.out::println);
```
